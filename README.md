# Agenda Telefônica em Python

Projeto de agenda telefônica em modo console com estrutura modular e persistência em arquivo texto.

## Funcionalidades

- Cadastro de contatos
- Listagem geral
- Pesquisa por nome
- Edição de registros
- Exclusão individual
- Exclusão total
- Armazenamento em arquivo `.txt`

## Estrutura

- `main.py` → controle de fluxo e menu principal
- `operacoes.py` → funções responsáveis pelo CRUD
- `dados.txt` → gerado automaticamente para armazenamento dos registros

## Conceitos Aplicados

- Listas e dicionários
- Modularização
- Manipulação de arquivos
- Validação de entrada
- Organização de código procedural

## Execução

```
python main.py
```
